Fake Currency Detection

A Fake Currency Detection System developed using Python, leveraging image processing techniques to distinguish between genuine and counterfeit notes. The system analyzes key features of the currency and achieves an accuracy of up to 81% on dataset of 2000-rupee Indian Currency, providing a reliable solution for detecting fake currency.
